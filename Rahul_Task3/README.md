# Password Generator

## Project Description
This is an advanced GUI-based Password Generator developed using Python and Tkinter. The application allows users to generate strong and secure passwords by selecting password length, character types, and excluding specific characters.

## Features
- Generate random secure passwords
- User-defined password length
- Include letters (a-z, A-Z)
- Include digits (0-9)
- Include symbols (!@#$%^&*)
- Exclude specific characters
- Input validation
- Minimum password length check (6 characters)
- Copy generated password to clipboard
- User-friendly GUI using Tkinter

## Technologies Used
- Python
- Tkinter
- Random Module
- String Module

## How to Run
1. Open the project folder.
2. Run the `main.py` file.
3. Enter the desired password length.
4. Select the character types to include.
5. Optionally enter characters to exclude.
6. Click the **Generate** button.
7. Click **Copy Password** to copy the generated password.

## Example Usage
- Password Length: 12
- Include Letters: Yes
- Include Digits: Yes
- Include Symbols: Yes
- Exclude Characters: O0l1
